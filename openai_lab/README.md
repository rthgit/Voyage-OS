# OpenAI Agent Lab

This project is a test bed for the OpenAI Agents SDK.

## Setup

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Set API Key**:
    You need an OpenAI API Key. Export it in your terminal:
    - **Windows (PowerShell)**: `$env:OPENAI_API_KEY="your-key-here"`
    - **Mac/Linux**: `export OPENAI_API_KEY="your-key-here"`

3.  **Run the Agent**:
    ```bash
    node triage_agent.mjs
    ```

## Files
- `triage_agent.mjs`: The "Language Triage Agent" example.
