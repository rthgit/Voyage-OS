## ğŸš€ Quick Start (Local Development)

### 1. Prerequisites
*   **Node.js** (v18 or higher) installed.
*   **OpenAI Account** (Plus/Pro) with **Developer Mode** enabled in ChatGPT Settings.

### 2. Installation
Open your terminal in the `APPGPT` folder:

```bash
# 1. Install Backend Dependencies
cd travel_mcp
npm install

# 2. Install Frontend Dependencies & Build
cd ../travel_ui
npm install
npm run build
```

### 3. Running the App
You need to run the server and expose it to the internet (Tunneling).

**Step A: Start the Server**
```bash
cd travel_mcp
node index.js
# Server runs on http://localhost:3001
```

**Step B: Start the Tunnel (in a new terminal)**
This makes your local server accessible to ChatGPT.
```bash
cd travel_mcp
lt --port 3001 --subdomain travel-os-test-mcp
```
*Note: Keep both terminals open!*
